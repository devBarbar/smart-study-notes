## Checklist

- Pass Reanimated shared/derived values directly to Skia props.
- Use Skia `interpolateColors` for animated colors.
- Keep high-frequency motion on the UI thread.
- Use path hooks for path interpolation or mutation.
- Use buffer hooks for animated atlas-like arrays.
- Keep gesture target semantics accessible in React Native views around the canvas.
